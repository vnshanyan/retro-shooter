# Retro Browser Games

A small collection of self-contained, single-file browser games. No build step, no
dependencies — just open the HTML file in a browser.

## Games

### 🔫 Retro Shooter (`shooter.html`)
A 2D top-down retro arcade shooter.

- **Move**: Arrow keys / WASD
- **Aim**: Mouse
- **Shoot**: Click (hold to keep firing)

Fight through 5 escalating waves of enemies (grunt, fast, tank), each spawning from
random angles and chasing you down. Procedural pixel-art sprites, screen shake, muzzle
flash, and a menu → play → level-complete → victory / game-over flow. Unlimited ammo
with a fire-rate cooldown.

### ⭕ Tic Tac Toe (`index.html`)
Classic tic-tac-toe with two modes:

- **2 Players** — local hot-seat
- **Vs Computer** — an unbeatable minimax AI

Includes a running scoreboard and highlighted winning line.

## Running

Open either file directly in any modern browser (double-click, or use a local static
server). Everything is inlined in one HTML file per game.
